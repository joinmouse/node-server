# markdown介绍

markdown是一种简单的标记语言，这些标记和html的一些标签一一对应，可以通过一些转换库将markdown与html相互转换

常见的标签如下：
## 标题

# 一级标题 （h1）
## 二级标题  (h2)
### 三级标题  (h3)

## 列表
— 无序列表
— 无序列表
— 无序列表

1. 有序列表
2. 有序列表
3. 有序列表
